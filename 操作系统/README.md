# 操作系统

这个目录用于沉淀操作系统安全基础与进阶内容，重点放在系统原理、日志、权限、进程、持久化和排障。

## 建议拆分方向

- Linux 基础与权限模型
- Linux 进程、服务与启动机制
- Windows 账户、服务、注册表、计划任务
- 事件日志与系统审计
- 主机入侵痕迹排查
- 常见持久化方式理解

## 推荐先学什么

1. Linux 文件权限、sudo、systemd、journalctl
2. Windows 事件日志、服务、计划任务、PowerShell
3. 进程树、网络连接、启动项排查
4. 把主机日志和应急响应串起来

## 适合沉淀的内容

- Linux 排查命令备忘录
- Windows 事件 ID 笔记
- 入侵痕迹排查 checklist
- 常见持久化手法的防守理解
