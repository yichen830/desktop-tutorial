# 易宸安全手册

一个面向中文网安学习者，尤其是希望从“会看漏洞”走向“会做体系化安全工作”的进阶仓库。

这个仓库不提供攻击利用代码，不提供针对未授权目标的操作说明，重点放在：

- 防御研究
- 资产与暴露面治理
- 检测与响应
- 安全工程化
- 中间件与应用安全
- 应急分析与日志研判

## 仓库定位

很多安全学习资料停留在“这个 CVE 怎么复现”，但真正往高阶走，能力结构会明显变化：

- 从单点漏洞，转向资产面和攻击面理解
- 从只会看公告，转向版本、配置、暴露面、日志的综合研判
- 从单个 PoC，转向检测、缓解、修复、复盘
- 从工具使用者，转向规则、流程和自动化的建设者

这个仓库就是围绕这些能力搭的。

## 你能在这里学到什么

### 1. 中间件漏洞防御研究

位于 `security-research/`，当前包含：

- Tomcat Ghostcat 防御分析
- Confluence OGNL 注入防御分析
- Struts2 文件上传路径穿越防御分析
- 一个基于资产清单的 Python 排查脚本

### 2. 学习路线与能力地图

位于 `learning-path/`，包括：

- 高阶学习路线图
- 蓝队能力进阶建议
- 检测工程基础框架

### 3. 实战导向的专题文档

位于 `playbooks/`，包括：

- 威胁狩猎思路
- 应急响应 checklist
- Web 与中间件基线排查思路

### 4. 合规工具

位于 `tools/`，只收录偏防守与审计用途的脚本，例如：

- 头部与暴露面审计
- IOC 时间线整理
- 版本与资产核查

## 推荐学习顺序

1. 先看 [learning-path/00_高阶网安学习路线图.md](./learning-path/00_高阶网安学习路线图.md)
2. 再看 [learning-path/01_蓝队与防御能力地图.md](./learning-path/01_蓝队与防御能力地图.md)
3. 然后进入 [security-research/README.md](./security-research/README.md)
4. 再结合 [playbooks/00_威胁狩猎实战思路.md](./playbooks/00_威胁狩猎实战思路.md) 和 [playbooks/01_应急响应清单.md](./playbooks/01_应急响应清单.md)
5. 最后把 `tools/` 里的脚本接到自己的资产盘点或日志分析流程里

## 适合谁

- 想从脚本小子成长为蓝队/安全研究/安全工程方向的人
- 已经会基础漏洞复现，但缺少体系化方法的人
- 想沉淀自己的安全知识库和检查脚本的人
- 想把安全学习做成长期仓库的人

## 安全与边界

本仓库仅用于授权环境下的安全学习、防御研究、资产排查、日志分析和修复验证。请勿将仓库内容用于未授权测试、攻击或破坏活动。
